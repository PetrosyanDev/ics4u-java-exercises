# Reflection: Java basics

## Data Types

-   Integers: `byte`, `short`, `int`, `long`
-   Floating-point: `float`, `double`
-   Characters: `char`
-   Text: `String`
-   True/False: `boolean`

```java
int x = 10;
double price = 19.99;
char letter = 'A';
boolean isOn = true;
```

---

## Expressions & Operators

-   Combine values, variables, and operators.
-   Follow precedence rules.
-   Integer division truncates results.

```java
int a = 7 / 2;      // 3
double b = 7.0 / 2; // 3.5
```

---

## Variables

-   A variable is a named storage for data.
-   It holds one value at a time.

```java
int age = 17;
String name = "Erik";
```

---

## Printing Output

-   `print()` → stays on same line
-   `println()` → goes to new line
-   `printf()` → allows formatted output

```java
System.out.print("Hi ");
System.out.println("there");
System.out.printf("Age: %d", age);
```
